# Алсофт Dashboard - React версия

React версия дашборда с использованием Vite для сборки.

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте браузер:
```
http://localhost:5173
```

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Header.jsx      # Шапка с метриками
│   ├── Tabs.jsx        # Навигационные вкладки
│   ├── Overview.jsx    # Вкладка "Обзор"
│   ├── Financial.jsx   # Вкладка "Финансы"
│   ├── Product.jsx     # Вкладка "Продукт" с roadmap
│   ├── Settings.jsx    # Вкладка "Настройки"
│   ├── RevenueChart.jsx # График выручки
│   ├── RoadmapModal.jsx # Модальное окно редактирования roadmap
│   └── SaveIndicator.jsx # Индикатор сохранения
├── context/
│   └── StateContext.jsx # Контекст для управления состоянием
├── App.jsx             # Главный компонент
├── main.jsx            # Точка входа
└── index.css           # Глобальные стили
```

## Особенности

- ✅ Полная функциональность оригинального дашборда
- ✅ Интеграция с Chart.js для графиков
- ✅ Редактирование roadmap с модальными окнами
- ✅ Автоматическое сохранение в API
- ✅ Адаптивный дизайн
- ✅ Темная тема

## API интеграция

Все данные синхронизируются с бекендом через `/api` endpoints:
- `/api/state` - состояние дашборда
- `/api/roadmap` - элементы roadmap
- `/api/reset` - сброс данных
- `/api/export` - экспорт данных

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.
